# Contributing

Thanks for your interest in contributing!

## Development setup

- Install Node.js (LTS recommended)
- Install dependencies:
  - `npm install`

## Run locally

- `npm run dev`

## Quality checks

- Lint: `npm run lint`
- Typecheck: `npm run typecheck`

## Pull requests

- Keep changes focused and small when possible.
- Include a short description and screenshots for UI changes.
- Ensure `npm run lint` and `npm run typecheck` pass.

## Reporting issues

Please include:
- your OS + Node.js version
- steps to reproduce
- expected vs actual behavior
- an example `.ics` (redact personal data if needed)
