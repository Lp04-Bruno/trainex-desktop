# Code of Conduct

This project aims to be a welcoming and safe space for everyone.

## Be respectful

- Be kind and constructive.
- Assume good intent.
- Focus on ideas and code, not on people.

## Unacceptable behavior

- Harassment, discrimination, or hate speech
- Sexualized language or unwanted sexual attention
- Threats, doxxing, or encouraging harm

## Enforcement

If something happens that violates these expectations, please contact the maintainer(s) privately.

Maintainers may remove content, close discussions, or block accounts to keep the project safe.
